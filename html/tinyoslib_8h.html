Program *  prog,

)

Try to reclaim the arguments of a process.

Given a procinfo object returned by a OpenInfo stream, if this process was executed by Execute, try to reclaim the arguments passed to it.

If the process was not executed by Execute, or if its arguments were too long, this effort may fail, in which case -1 is returned.

Else, the pinfo argument buffer defined by (pinfo->argl, pinfo->args) is parsed. Let it contain N arguments. Then,

  • if prog is non-NULL, *prog is filled with a pointer to the function executed by Execute
  • if argv is non-NULL, it is assumed that it points to an array of strings, of size argc. Then, min(argc, N) locations of this array are initialized from the arguments of pinfo.
  • finally, N is returned.
Parameters
pinfothe procinfo object to process.

if pinfo is successfully parsed, the number of arguments is returned, else -1 is returned.
